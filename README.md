I present to you another DIY project.

If for one reason or another you're using old oil heaters, this article is for you.

The thermostatic system in old heaters, like in an iron, regulates the temperature of the heater itself, not the air. It's inexpensive and durable, but getting the right temperature in the room is a challenge. This thermostat should handle it.
